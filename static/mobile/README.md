## 截图要求

- **格式**: PNG 或 JPG
- **尺寸**: 建议宽度 300-400px，保持手机屏幕比例
- **质量**: 清晰度高，无模糊
- **内容**: 展示主要功能界面

## 如何添加截图

1. 将截图文件放入此目录 (`public/screenshots/mobile/`)
2. 截图会自动在移动端页面中显示
