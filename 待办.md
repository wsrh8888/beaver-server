# Beaver Server 待办清单

| 优先级 | 分类 | 问题 | 说明 |
|--------|------|------|------|
| P0 | 架构 | datasync 缺增量同步 | 16 个接口全是全量拉取，用户量大首次登录必然卡顿，需改为基于 seq/version 的增量接口 |
| P0 | 消息可靠性 | 群聊消息 seq 并发竞态 | 当前用 MAX(seq)+1，高并发下会产生重复 seq，需改为 Redis INCR 原子生成 |
| P0 | 消息可靠性 | 缺已送达回执（Delivered ACK） | 目前只有已读，没有"已送达"状态，用户发出消息无法感知对方是否收到 |
| P0 | 消息可靠性 | 离线消息无持久化补偿 | 离线时依赖 FCM/APNs，推送失败或用户禁用推送则消息丢失，需增加离线消息 DB 拉取补偿 |
| P1 | 消息功能 | 缺消息编辑接口 | 数据模型已有 status 字段，但 chat_api 和 chat_rpc 均无 editMessage 接口，补接口成本低 |
| P1 | 消息功能 | 缺消息 Reaction（表情回应） | 无任何 reaction 接口，Slack/Discord/飞书/钉钉标配，OpenIM v3 已支持 |
| P1 | 消息功能 | 群消息缺已读人数/已读列表 | updateReadSeq 是会话级，无"哪些人读了这条消息"接口，B 端场景强需求 |
| P1 | 用户功能 | 用户搜索未暴露 HTTP 接口 | user_rpc 有 SearchUser，但 user_api 无对应路由，用户无法搜索陌生人加好友 |
| P1 | 安全 | JWT 使用 HS256 共享密钥 | 所有服务共用同一密钥，任一服务泄露全线失守，建议改为 RS256 非对称签名 |
| P1 | 安全 | 缺 API 限流 | 消息发送等核心接口无速率限制，存在刷屏/DoS 风险 |
| P2 | 群组功能 | 群公告无多条历史管理 | 公告混在 updateGroupInfo 里，只能存一条最新，无独立的增删改查接口 |
| P2 | 群组功能 | 缺群成员搜索接口 | getGroupMembers 全量拉取，千人大群无法按名称搜索成员 |
| P2 | 群组功能 | 缺邀请链接/邀请码接口 | 无法生成群邀请链接，入群控制不完整 |
| P2 | 消息功能 | 缺消息草稿多端同步 | 无 saveDraft/getDraft 接口，切换会话草稿丢失且不跨端同步 |
| P2 | 运营能力 | 缺管理员主动推送系统消息 | 后台无法向指定用户或全体用户推送运营通知/公告，运营场景无法支持 |
| P2 | 通知功能 | 通知缺分类静默策略 | 有分类概念但无"按分类设置免打扰"接口，如"点赞通知静默但系统通知提醒" |
| P2 | 好友功能 | 好友备注修改无多端同步 | datasync 无 getSyncFriendNotices，备注改了其他设备不会同步 |
| P2 | 基础设施 | 缺用户在线状态 HTTP 查询接口 | 无查询某用户在线状态的接口，业务层无法对外暴露在线状态 |
| P3 | 安全 | 消息内容明文存储 | Msg 字段存 JSON 明文，无端到端加密，DBA 可直接读取消息内容 |
| P3 | 安全 | 缺二步验证（2FA/TOTP） | 无二步验证接口，账号安全依赖单一密码 |
| P3 | 扩展性 | 文件存储强依赖七牛云 | 无存储抽象层，无法对接 S3/MinIO/阿里云 OSS，国外用户无法使用 |
| P3 | 群组功能 | 缺群文件管理 | 只能发文件消息，无群文件列表/删除/重命名等管理接口 |
| P3 | 通话功能 | 缺通话录制接口 | LiveKit 原生支持录制，但未封装对应接口 |
| P3 | 可观测性 | 缺链路追踪 | 无 OpenTelemetry/Jaeger 集成，分布式问题排查困难 |
| P3 | 测试 | 无单元测试 | test 目录为空，贡献者无法安全提交 PR |
| P3 | 文档 | 无 API 文档 | 无 Swagger/OpenAPI 文档，接入方只能读源码 |

## 大模块规划

| 优先级 | 模块 | 说明 |
|--------|------|------|
| 高 | 开放平台智能机器人 | 在现有群机器人基础上接入 AI 能力，第三方开发者可接入任意 AI 模型，提供上下文透传、流式回复支持 |
| 高 | AI 助手 | 系统内置官方 AI 对话能力，部署者配置自己的 API Key，类似飞书 My AI |
| 中 | 圈子/社区模块 | 话题空间，成员发帖+评论+点赞，可邀请加入，复用朋友圈点赞评论底子 |
